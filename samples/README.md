# Sample Documents

This directory contains shared Markdown files for manual testing.

- `basic.md` checks basic English text and lists.
- `japanese.md` checks Japanese display.
- `code-blocks.md` checks fenced code blocks.
- `math.md` checks inline and display math in paragraphs, quotes, lists, and tables.
- `mermaid.md` checks Mermaid fenced block fallback display and source copying.
- `long-form.md` checks longer reading flows, heading navigation, zoom, themes, and live reload.

Keep these files small, readable, and useful for repeated UI checks.
