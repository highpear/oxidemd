## Product Vision

OxideMD is a fast, local-first Markdown viewer designed not only for developers but also for non-technical users.

The goal is to provide a simple, intuitive, and high-performance experience for reading Markdown documents.

---

## Core Principles

- Fully local (no cloud dependency)
- Fast and lightweight
- Simple and intuitive UI
- Designed for non-engineers as well as developers
- Clean and distraction-free reading experience

---

## Planned Features

### Core Features
- Fast Markdown rendering
- Local file viewing
- Real-time preview (file watching)

### Usability Features
- Familiar UI for non-technical users
- Tab-based multi-file viewing
- Table of contents (TOC)
- Smooth navigation

### Advanced Features
- Math rendering (LaTeX)
- Mermaid diagram support
- Advanced search functionality
- Syntax highlighting

---

## Non-Goals (for now)

- Full Markdown editor
- Cloud sync
- Plugin ecosystem (initially)
